# Idle Colors
Idle Colors is an incremental game meant to be pleasant to just sit around and watch, and fun to engage with.

## Technologies Used:
Godot (GDScript, GDShader)

## Instructions:
- Score shapes to earn points. Place them in the hatch, then hit the button to score. Or let them pile up until they hit the hatch button themselves.
- 'C' activates the hatch manually
- Clicking the scoreboard opens the upgrade screen
